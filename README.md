# 🤖 شات بوت الذكاء الاصطناعي (AI Chatbot)

شات بوت ذكاء اصطناعي متكامل مبني باستخدام لغة Python وإطار عمل Streamlit. يوفر واجهة مستخدم احترافية وتفاعلية تتيح لك التواصل مع أحدث نماذج الذكاء الاصطناعي (مثل GPT-4o و Gemini و Claude) مع إمكانية رفع الملفات واستخراج المعلومات منها.

## ✨ المميزات

* **دعم النماذج المتعددة:** إمكانية اختيار النموذج المفضل لديك من بين نماذج OpenAI و Google و Anthropic.
* **تحليل الملفات (RAG):** دعم رفع وقراءة ملفات PDF, Word (DOCX), CSV, و TXT، والبحث في محتواها للإجابة على الأسئلة.
* **ذاكرة المحادثة:** يحتفظ البوت بسياق المحادثة السابقة لتوفير إجابات دقيقة ومتسقة.
* **تخصيص شخصية البوت:** إمكانية تعديل الـ System Prompt لتحديد طريقة استجابة البوت.
* **واجهة مستخدم عصرية:** تصميم جذاب مع دعم للوضع الليلي، ألوان متناسقة، ومؤشرات بصرية للحالة.
* **تتبع الاستهلاك:** عرض إحصائيات حول عدد الرسائل والملفات المرفوعة.

## 🛠️ المتطلبات الأساسية

* Python 3.9 أو أحدث
* مفتاح API للنموذج الذي ترغب في استخدامه (مثل OpenAI API Key)

## 🚀 طريقة التثبيت والتشغيل

1. **تحميل الكود:**
   قم بتحميل جميع ملفات المشروع إلى مجلد واحد.

2. **تثبيت المكتبات المطلوبة:**
   افتح موجه الأوامر (Terminal/CMD) في مجلد المشروع وقم بتشغيل الأمر التالي:
   ```bash
   pip install -r requirements.txt
   ```

3. **تشغيل التطبيق:**
   قم بتشغيل التطبيق باستخدام أمر Streamlit:
   ```bash
   streamlit run app.py
   ```

4. **الاستخدام:**
   * سيتم فتح التطبيق في المتصفح الافتراضي (عادة على الرابط `http://localhost:8501`).
   * أدخل مفتاح الـ API الخاص بك في الشريط الجانبي.
   * اضغط على زر "تحقق من مفتاح API".
   * يمكنك الآن البدء في الدردشة أو رفع الملفات!

## 📁 هيكل المشروع

* `app.py`: ملف الواجهة الرئيسية باستخدام Streamlit.
* `chatbot_engine.py`: محرك الشات بوت والمسؤول عن التواصل مع الـ APIs الخاصة بالنماذج.
* `file_processor.py`: مسؤول عن معالجة الملفات (PDF, DOCX, CSV) وبناء قاعدة المعرفة باستخدام FAISS و LangChain.
* `config.py`: ملف الإعدادات الرئيسية (النماذج المدعومة، إعدادات الملفات، وغيرها).
* `requirements.txt`: قائمة بجميع المكتبات المطلوبة لتشغيل التطبيق.
* `.streamlit/config.toml`: إعدادات تخصيص مظهر وتكوين Streamlit.

## 📝 ملاحظات هامة

* **الأمان:** لا يتم حفظ مفاتيح API الخاصة بك على أي خادم؛ يتم تخزينها مؤقتًا في جلسة المتصفح الخاصة بك فقط.
* **الملفات المرفوعة:** يتم حفظ الملفات المرفوعة وقاعدة المعرفة محليًا في مجلدي `uploads` و `data` داخل مسار المشروع لتسهيل عملية البحث السريع.
